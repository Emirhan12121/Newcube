Rehber — Nasıl Giriş Yapılır?

  1. Minecraft Java 1.20.x sürümünü aç.
  2. Multiplayer → Add Server deyip adresi gir: play.skyhavan.com.
  3. Sunucuya bağlan, ilk girişte /register şifre şifre ile kayıt ol.
  4. Sonraki girişlerde /login şifre yaz.
  5. /spawn başlangıca döner, /sethome ve /home ile eve ışınlan.
  6. İlk ada için /is create yaz; arkadaş eklemek için /is invite.
  7. Para için market kur, /sat, /ah ve /ticaret kullan.
