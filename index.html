Rehber — Nasıl Giriş Yapılır?

  1. Minecraft Java 1.20.x sürümünü aç.
  2. Multiplayer → Add Server de.
  3. Sunucu adresi: play.skyhaven.gg
  4. Bağlan ve /register şifre şifre ile kayıt ol, /login şifre ile giriş yap.
Skyblock’ta para nasıl kazanılır?

Adanda market kur, oyuncularla ticaret yap (/ticaret), eşyalarını sat (/sat), ihaleye koy (/ihale).

Yeni ada ve üye yönetimi

/is create ile ada oluştur; /is invite ile üye davet et; /is settings ile izinleri düzenle.

Hızlı komutlar

/spawn başlangıç, /sethome & /home ev noktası, /tpa ışınlanma isteği.
