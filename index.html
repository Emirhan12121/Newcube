Rehber — Nasıl Giriş Yapılır?

  1. Minecraft Java 1.20.x sürümünü aç.
  2. Multiplayer → Add Server de.
  3. Sunucu adresi: play.skyhaven.gg
  4. Bağlan ve /register şifre şifre ile kayıt ol, /login şifre ile giriş yap.
